پایانِ کابوسِ
کانفیگ‌های دستی

باگ‌های اعصاب‌خردکن YAML-Hell را دور بزنید. یک استکان چای لب‌سوز بریزید و بگذارید دایرکتوری‌ها، کانتینرها و شبکه‌های ایزوله شما به صورت خودکار با قدرت Go متولد شوند.

crazyDev-installer ~ bash
❯ git clone https://github.com/UcrazyNess/crazyDev.git
❯ cd crazyDev
❯ go run main.go
دستور کپی شد!

ویژگی‌های برجسته

تمام دغدغه‌های زیرساختی شما در یک هسته مینیمال و قدرتمند

داربست‌بندی پویا

تعریف سناریوهای مرحله به مرحله برای ساخت پوشه‌ها، تزریق کدهای پایه (Template) و نصب وابستگی‌ها به صورت کاملا خودکار و بدون دخالت دست.

قاتلِ YAML

طراحی بصری و تولید آنی فایل‌های پیچیده docker-compose با پشتیبانی کامل از شبکه‌های اختصاصی و IPهای ثابت.

اجرای زنده (Remote)

اتصال مستقیم و امن پنل وب به ترمینال سرور فیزیکی و پیاده‌سازی معماری تنها با یک کلیک به همراه استریم زنده و بلادرنگ لاگ‌ها.
