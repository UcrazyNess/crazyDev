# crazyDev
پروژه crazyDev یک استودیوی معماری و ارکستریشن زیرساخت است که باگ‌های اعصاب‌خردکنِ کانفیگ دستی را دور می‌زند و کارهای خسته‌کننده دواپس را اتوماتیک می‌کند.

> باگ‌های اعصاب‌خردکنِ کانفیگ دستی و YAML-Hell را دور بزنید. یک فنجان قهوه داغ بریزید و بگذارید دایرکتوری‌ها، کانتینرها و شبکه‌های ایزوله شما به صورت خودکار متولد شوند.

## 🚀 داستان این ابزار چیه؟

پروژه **crazyDev** با یک هدف مشخص متولد شد: **حذف کارهای تکراری و مستعد خطای انسانی در ستاپ زیرساخت.**
فرقی نمی‌کند در حال توسعه و اتصال ده‌ها میکروسرویس بک‌اند به یکدیگر باشید، یا بخواهید شبکه‌های ایزوله و محیط‌های آزمایشگاهی را برای سناریوهای Red Teaming و تست نفوذ در کسری از ثانیه اسپاون کنید؛ این سیستم تمام دغدغه‌های زیرساختی را با قدرت‌گیری از یک هسته مینیمال Go و یک رابط کاربری هوشمند هندل می‌کند.

## ✨ فیچرهای خفن (Features)

* 🧱 **اسکفولدینگ داینامیک (Framework Engine):** تعریف سناریوهای Step-by-Step برای ساخت پوشه‌ها، تزریق کدهای تمپلیت و نصب پکیج‌ها.
* 🐳 **قاتلِ YAML (Compose Builder):** طراحی بصری و تولید آنی فایل‌های `docker-compose.yml` با پشتیبانی کامل از شبکه‌های کاستوم (Subnet, Gateway) و IPهای استاتیک.
* 📦 **رجیستری میکروسرویس‌ها:** ذخیره و بازخوانی سریع بلاک‌های کانفیگ داکر برای سرویس‌های پایه (وب‌سرورها، دیتابیس‌ها و...).
* 🔥 **اجرای لایو (Remote Execution):** اتصال مستقیم پنل وب به ترمینال سرور فیزیکی و پیاده‌سازی معماری با یک کلیک به همراه استریم زنده لاگ‌ها.
* 💻 **رابط دوگانه (CLI + Web UI):** هم از طریق محیط گرافیکی مدرن (Dark/Cyberpunk Vibe) و هم از طریق ترمینال (CLI) در دسترس شماست.

## 🛠 نصب و ستاپ (Quick Start)

### پیش‌نیازها

* `Go 1.20+`
* `Docker & Docker Compose`

### ۱. کلون کردن مخزن

```bash
git clone [https://github.com/yourusername/crazyDev.git](https://github.com/UcrazyNess/crazyDev.git)
cd crazyDev

```

### ۲. اجرای نسخه وب (Web Server)

برای لانچ کردن استودیوی گرافیکی روی پورت لوکال:

```bash
go mod tidy
go run main.go
# 🌐 Now open http://localhost:5500 in your browser

```

### ۳. بیلد کردن نسخه خط فرمان (CLI)

اگر ترجیح می‌دهید دستتان از روی کیبورد برداشته نشود:

```bash
go build -o crazydev ./cli/main.go
sudo mv crazydev /usr/local/bin/

# تست نصب:
crazydev --help

```
